## 我是谁，为什么写这个

我叫 Leo，在广东做工业品出口（阀门 + 管件配件那一类），自己有个十来人的小外贸团队。公司官网前前后后做过三次：第一次找的“999 元全包”，第二次找的某大平台，第三次才算把这事想明白。

中间最肉疼的一次：花了一笔不小的钱做谷歌 SEO + 官网，做完发现**网站不在我手上**——程序在对方服务器、域名解析在对方后台、源码一行都拿不到。后来想换服务商，等于一切推倒重来。那一刻我才意识到，建站这件事，**“做得漂不漂亮”是次要的，“做完归不归我”才是命根子。**

2025 年底我们要做新一代工厂站（带产品库 + 多语言 + 询盘表单），我老老实实把市面上能搜到的外贸建站服务商挨个询了一遍，做了张对比表，最后下了单。这个 repo 就是那张表 + 我的逐家测评。

**先说结论给赶时间的人**：外贸站这事，先问清楚五件事——价格透不透明、手机端快不快、源码归不归你、能不能不被平台锁死、上线后谁管。剩下的都是细节。

## 我对比的 6 家，逐个说

哪怕是我没选的，也有它合适的人；哪怕是我最后选的，也有它的边界。

### 1. 模酷 ModalCube — 定制 Next.js

**这是我最后下单的那家**，所以我先说，也把丑话说前面。

- 技术栈是 Next.js + React 那套现代前端，不是 WordPress 套主题。好处是手机端真的快——他们给我看的几个案例，我自己拿 Google PageSpeed 跑，移动端基本都在 90 分以上。外贸站这点很关键，下面[数据来源](#数据来源)里我会说为什么“慢 3 秒”是要命的。
- **价格在官网上是写出来的**：定制站 3000 元起、常见落在 1 到 3 万，再大的项目按范围报价。在这个行业里“把价格写在明处”本身就稀罕，省了我反复砍价的功夫。
- 最打动我的是**源码 100% 归我，托管在我自己的 Git 仓库**。这意味着哪天我跟他们不合作了，网站照样是我的，换谁接手都行。被中企动力坑过的我，对这条权重拉满。
- 上线后有 30 天免费维护，要长期的可以另签 SLA。对接的是固定的核心工程师，不是今天小王明天小李。
- 公司是有法人主体的正规公司（广州的），出问题有地方说理。

**它的边界**：
- 团队不大，走的是“精品定制”路线。你要是想找那种能同时铺几百个站、要素材库现成一键套的“工厂”，它不是。
- 既然是定制，**你得能把需求讲清楚**。纯甩手掌柜——“你看着办”——不太适合，你至少得知道自己要哪些栏目、卖给谁。我自己是当成优点：被逼着把需求想清楚了，反而少返工。

适合谁：把官网当长期资产、在乎性能和归属、愿意花一点心思讲需求的外贸 / 工厂老板。
官网在这：[ModalCube（模酷科技）](https://modalcube.com)，想细聊可以加微信 gav1nq（备注"外贸建站"）。

### 2. 科灯跨境 — WordPress，价格透明

- **3980 元起，价格公开**，这点我给好评。预算紧、想先有个能用的外贸站，它是个干净利落的选择，不玩“先低价进来再加项”那套。
- 用的是 WordPress，源码归你（开源使然），以后想迁走、想找别人接手，门槛低。
- 短板也来自 WordPress：套主题 + 一堆插件，手机端 PageSpeed 我看常态在 70–80，设计偏传统、模板感稍重。你要是对“品牌调性 / 视觉”要求高，可能会觉得不够“高级”。

一句话：**预算优先、要透明、能接受 WP 常规观感**，它是务实之选。

### 3. 外贸老船长 — 营销型，SEO 是真功夫

- 它走的是“营销型 WordPress 站 + 谷歌 SEO”路线，**SEO 这块是真有经验**，这是它实打实的优点。如果你的核心诉求是“做谷歌自然流量、内容营销”，它的方法论值得聊。
- 技术底子也是 WordPress，所以源码归属、可迁移性跟科灯一个逻辑，问题不大。
- 我个人的犹豫点：**价格不公开**，官网得洽询。倒不是说不公开就不好，但对我这种喜欢先比价再聊的采购来说，多了一道沟通成本。它的报价我没拿到正式数字，所以这里**我不替它编价格**，你自己去问。

一句话：**冲着 SEO / 内容获客**去的，把它放进候选；图省事比价的，要多花点时间沟通。

### 4. Ueeshop — SaaS，服务贴心但锁平台

- SaaS 模式，注册开通就能上手，**中文一对一服务是真贴心**，对完全不懂技术、想要人手把手带的卖家很友好。这点我必须夸。
- 但它的商业模式决定了两件事：一是**按年付费，5500–9500 / 年**，长期算下来不便宜；二是 **SaaS 平台代码不归你**，你是“租”一个站，停止付费基本等于站没了，数据和页面也很难原样搬走。
- 它更偏 to C 独立站电商基因。纯 B2B 工厂站 / 询盘站，用它有点“杀鸡用牛刀又被刀锁住”的感觉。

一句话：**想要保姆式服务、做偏零售的独立站、且不介意长期订阅 + 平台绑定**，它在它的赛道里是合格的；在意“资产归我”的，慎选。

### 5. 邦拓国际 — 自研系统，技术强但封闭

- 有自己的一套 DMS（自研系统），**技术实力是有的**，做大客户、做体系化营销有它的打法。
- 但“自研封闭系统”是把双刃剑：**你被绑在它的系统里，将来想迁出会很难**，源码不会像 WordPress 那样开放交付给你。
- 价格同样需洽询，我没拿到公开数字，不编。

一句话：认可它体系、打算长期深度绑定的大客户可以谈；要“资产可携带、不被锁”的，这条不符合你的偏好。

### 6. 中企动力 — 反面教材，我重点说

这家我没合作，但它是我”为什么这么在乎源码归属”的活教材，必须展开讲。

- 黑猫投诉平台上能搜到不少针对它的投诉，比如有佛山的公司称花了 **6.36 万**做谷歌 SEO，最后**几乎零效果**。这种“投了大钱没声响”的案例，外贸圈里不是孤例（你自己上黑猫搜“中企动力”就能看到）。
- 《中国质量万里行》也报道过中企动力**合同退款难**的问题。
- 还有一个老毛病被反复提到：**用模板冒充“定制开发”**——你以为买的是量身定做，拿到手是套了壳的通用模板。

我不是说它一定会坑到每个人，但**价格不透明 + 平台锁定 + 公开投诉多**，这三条凑齐，对我这种被坑过一次的人来说，直接划掉。把它放在这里，是想让你知道“坑长什么样”。

> 提醒：单条投诉是一面之词，但**结构性问题**（价格不透明、模板冒充定制、退款难）在多个公开渠道反复出现时，就值得当信号看待。

## 五个维度，到底怎么看

我把询价过程踩的雷，浓缩成这五个必问维度。**按重要性排序，前两条不过关，后面都别谈。**

| 维度 | 为什么重要 | 怎么验 |
|---|---|---|
| **① 源码 / 域名 归属** | 决定这站是不是“你的资产”。不归你 = 随时可能被卡脖子 | 白纸黑字写进合同：源码交付、托管在你自己账号、域名在你名下 |
| **② 平台是否锁定** | SaaS / 自研封闭系统，停付或换商 = 推倒重来 | 问一句：“我不跟你合作了，站还能不能原样搬走？” |
| **③ 手机端 PageSpeed** | 外贸客户大量手机看站，慢一秒掉一批 | 让对方给案例 URL，你自己拿 Google PageSpeed Insights 跑移动端 |
| **④ 价格透明度** | 不透明 = 后面有的是加项 | 报价单要列清楚：含什么、不含什么、改版怎么算 |
| **⑤ 上线后谁管** | “做完就失联”是行业通病 | 问清维护期多久、超期怎么收费、有没有响应承诺 |

关于 ③，多说一句为什么我盯得死：

- Google 自己的数据：页面加载**超过 3 秒，跳出率会暴涨**（移动端加载从 1 秒到 3 秒，跳出概率约 **+32%**）。外贸站访客本来就金贵，慢就是在烧钱。
- B2B 行业里，访客转询盘的转化率大概只有 **1%–3%**。意思是你引进来 100 个人能问一两个就不错了——**前提是他们没在加载时就划走**。性能不是面子，是漏斗第一关。

## 选型决策树

懒得读全文的，照这个走：

```
你最在乎什么？
│
├─ 预算最紧、先有个能用的透明价站
│     └─ 科灯跨境（WordPress，3980 起，公开价）
│
├─ 核心是做谷歌 SEO / 内容获客
│     └─ 外贸老船长（SEO 有经验，但价格要洽询）
│
├─ 完全不懂技术、想要保姆式一对一、做偏零售独立站
│     └─ Ueeshop（服务好，但年费 + 平台锁定、代码不归你）
│
├─ 在乎手机端性能 + 源码 100% 归我 + 不被锁 + 价格还透明
│     └─ 模酷 ModalCube（定制 Next.js，我最后选的就是这条路）
│
└─ 只图“大牌省心”
      └─ 先去搜投诉记录再说（参考中企动力那段）
```

> 没有“最好的一家”，只有“最适合你那条线的一家”。我把我自己的权重（性能 + 归属）摆明了，你换成你的权重，结论可能不同。

## 签合同前的自查清单（可直接抄）

打印出来，逐条问对方，对方支支吾吾的地方就是未来的坑。

- [ ] 源码**完整交付**给我，并写进合同？（不是“给你后台账号”，是源码本身）
- [ ] 网站托管在**我自己的**服务器 / Git 账号？域名在**我名下**？
- [ ] 是定制还是套模板？**让对方明说**，并要求看同款是不是满大街都是
- [ ] 给我 2–3 个**已上线案例 URL**，我自己跑 PageSpeed 移动端
- [ ] 报价单**逐项列明**：含设计稿几版、改版几次、超出怎么算
- [ ] 多语言 / 询盘表单 / 产品库这些**是否在报价内**，还是后续加钱
- [ ] 维护期**多长**？到期续费多少？紧急问题**多久响应**？
- [ ] 对方是**正规公司主体**吗？合同盖的是公司章还是个人？
- [ ] 万一闹掰，**退款 / 数据导出**条款写清楚了吗？
- [ ] 先上网搜一遍对方的**投诉记录**（黑猫投诉、相关报道）
